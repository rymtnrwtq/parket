# Структура проекта

```
site/
├── index.html                 # Главная страница сайта
├── css/                       # Директория со стилями
│   ├── README.md             # Документация CSS структуры
│   ├── main.css              # Главный файл (импортирует все модули)
│   ├── base.css              # Базовые стили и типография
│   ├── components.css        # Переиспользуемые компоненты
│   ├── animations.css        # Анимации и keyframes
│   └── utilities.css         # Утилитарные классы
└── Фото паркет/              # Директория с фотографиями продукции
```

## Преимущества модульной структуры CSS

### ✅ Организация
- Каждый файл отвечает за свою область
- Легко найти нужные стили
- Понятная структура для новых разработчиков

### ✅ Поддержка
- Изменения в одном модуле не влияют на другие
- Легче отлаживать проблемы
- Простое добавление новых стилей

### ✅ Производительность
- Браузер кэширует отдельные файлы
- Можно загружать только нужные модули
- Легко минифицировать для продакшена

### ✅ Масштабируемость
- Легко добавлять новые модули
- Можно создавать подкатегории
- Простая интеграция с препроцессорами (SASS, LESS)

## Как работать с CSS

### Добавление новых стилей

1. **Базовые стили** (шрифты, цвета, общие правила) → `base.css`
2. **Компоненты** (кнопки, карточки, модалы) → `components.css`
3. **Анимации** (переходы, keyframes) → `animations.css`
4. **Утилиты** (helper-классы) → `utilities.css`

### Пример добавления нового компонента

```css
/* В components.css */

/* Product Card */
.product-card {
    background: white;
    border-radius: 12px;
    box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease;
}

.product-card:hover {
    transform: translateY(-4px);
}
```
