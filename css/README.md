# CSS Structure

Эта директория содержит модульные CSS файлы для сайта Паркетной Фабрики.

## Структура файлов

### `main.css`
Главный файл, который импортирует все остальные модули. Подключается в `index.html`.

### `base.css`
Базовые стили:
- Типография (шрифты для body, заголовков)
- Глобальные стили

### `components.css`
Переиспользуемые компоненты:
- `.wood-pattern` - фоновый узор дерева
- `.hero-bg` - фон главного баннера
- `.modal-transition` - переходы для модальных окон
- `.shimmer` - эффект загрузки

### `animations.css`
Анимации и keyframes:
- `heroZoom` - анимация увеличения фона
- `placeholderShimmer` - анимация загрузки

### `utilities.css`
Утилитарные классы:
- `.scrollbar-hide` - скрытие полосы прокрутки (кросс-браузерное)

## Как добавить новые стили

1. Определите категорию стиля (base, components, animations, utilities)
2. Добавьте стили в соответствующий файл
3. Добавьте комментарий с описанием

## Порядок загрузки

Файлы загружаются в следующем порядке через `main.css`:
1. base.css
2. components.css
3. animations.css
4. utilities.css

Это обеспечивает правильный каскад и переопределение стилей.
